# The Investor's Playbook

A personal stock analysis playbook with 20 curated stocks across early-stage, growth, and established categories.

## Quick Deploy to Vercel (free)

1. Go to vercel.com and sign up with your GitHub or email
2. Click "Add New… → Project"
3. Click "Browse" and select this folder (or drag and drop the folder into Vercel)
4. Vercel auto-detects Vite — just click "Deploy"
5. In ~60 seconds you'll get a URL like `investors-playbook.vercel.app`

## Add to iPhone home screen

1. Open the Vercel URL in Safari on your iPhone
2. Tap the Share button (square with up arrow)
3. Scroll down, tap "Add to Home Screen"
4. Name it "Playbook" → Add

Now it launches like a real app, full-screen, no browser UI.

## Updating the app

When you want changes:
1. Ask Claude for an updated `App.jsx`
2. Replace `src/App.jsx` in your Vercel project (edit in Vercel's web UI or push to GitHub)
3. Vercel auto-rebuilds — your phone app updates in ~30 seconds

## Local development (optional)
